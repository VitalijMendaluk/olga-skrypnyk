# Olga Skrypnyk — Beauty Studio (сайт + Telegram-заявки)

Статичний двомовний (DE/UA) сайт + одна серверна функція, яка надсилає заявки з форм у Telegram.

```
index.html          — сам сайт
assets/             — фото, логотип, favicon
api/lead.js         — серверна функція: приймає заявку і шле в Telegram
.env.local          — токен бота (ЛОКАЛЬНО, ніколи не публікувати)
```

## Як це працює

Форма на сайті → `POST /api/lead` → функція бере токен зі **змінної оточення** → шле повідомлення боту.
**Токена в сторінці немає** — його не видно у вихідному коді сайту.

Повідомлення в Telegram виглядає так:

```
💌 Neue Terminanfrage

💅 Behandlung: Soft Volume 4–5D
💶 Preis: € 70
👤 Name: Anna Muster
📞 Telefon: +49 170 1234567
```

## Публікація на Vercel

1. Завантаж проєкт на Vercel (через GitHub або `npx vercel`).
   Фреймворк — **Other**. Vercel сам зрозуміє: `index.html` — сайт, `api/` — функції.

2. **Обов'язково** додай змінні оточення:
   Vercel → твій проєкт → **Settings → Environment Variables** → додай дві:

   | Name | Value |
   |---|---|
   | `TELEGRAM_BOT_TOKEN` | `8858606851:AAEuRYsKUE5HBAsif_edaYwWqkrnojRflY4` |
   | `TELEGRAM_CHAT_ID` | `-5168378976` |

   Постав галочки для **Production, Preview, Development**.

3. Натисни **Redeploy** (змінні підхоплюються лише при новому деплої).

4. Відкрий сайт → заповни форму → у групі «Olga_Skrypnyk_Заявки» має прийти повідомлення.

## Локальна перевірка (необов'язково)

Відкриття `index.html` подвійним кліком **не запустить** серверну функцію — форма покаже помилку.
Щоб перевірити локально, потрібен Node.js:

```bash
npm i -g vercel
cd olga-skrypnyk-site
vercel dev          # підхопить .env.local
```

## Важливо

- **`.env.local` не публікувати й не комітити** — там токен. Його вже додано в `.gitignore`.
- Якщо токен колись «засвітиться» — перевипусти його у **@BotFather → /revoke** і онови змінну на Vercel.
- **Бот має лишатися в групі.** Якщо його видалити — заявки не приходитимуть.
- Якщо групу колись оновлять до **супергрупи**, її ID зміниться (стане `-100…`).
  Тоді заявки перестануть приходити — треба буде оновити `TELEGRAM_CHAT_ID`.
